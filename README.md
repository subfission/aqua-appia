# Aqua Appia
Modular, sensor driven, and affordable irrigation.

## Sensors & Modules

- Ambient Temperature
- Humidity
- Soil Moisture
- Sunlight (avg. per day)
- Wind Speed
- Pressure
- Soil temp
- Soil Amendment Schedule
- Pest Laser
- Multi-Speed Fan Control 
- Multi-Position Vent Control
